CSS Animation Technique

/* Particles follow SVG path using offset-path */
.flow-particle {
  offset-path: path('M 130 90 C 180 90, 220 90, 270 90');
  offset-rotate: 0deg;
  animation: flowParticle 3s linear infinite;
}

@keyframes flowParticle {
  0% { offset-distance: 0%; opacity: 0; }
  5% { opacity: 1; }
  95% { opacity: 1; }
  100% { offset-distance: 100%; opacity: 0; }
}

Uses CSS offset-path for particle motion along bezier curves. Staggered animation-delay creates continuous flow effect.

Implementation Note

This is a demo loop showing the sequential reveal animation. In the actual app, the animation will be driven by user workflow state:

  • Completed phases show their icon (static, with gentle pulse)
  • Current phase shows animated particles (work in progress)
  • Future phases remain hidden until reached
